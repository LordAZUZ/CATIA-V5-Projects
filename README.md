CATIA V5 Projects

Overview

This repository contains my CATIA V5 design projects created to improve my mechanical design and CAD skills. The projects include 3D part modeling, assembly design, and technical drawings.

Skills Demonstrated

- 3D Part Design
- Assembly Design
- Drafting (2D Technical Drawings)
- Constraints and Assembly Management
- Parametric Modeling
- Design Validation

Software

- CATIA V5

Repository Structure

- Parts/ – Individual 3D parts (.CATPart)
- Assemblies/ – Assembly files (.CATProduct)
- Drawings/ – 2D drawings (.CATDrawing)
- Images/ – Screenshots and rendered images

- Demonstrate proficiency in CATIA V5 for automotive and industrial applications.
